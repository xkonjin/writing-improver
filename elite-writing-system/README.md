## Key Principles

### From Kolmogorov Clarity

- Compress to irreducible minimum
- Every word must be necessary
- Start with conclusion
- Delete all hedging

### From Master Writers

- **Paul Graham**: Write like you talk, but impossibly perfectly
- **Ben Thompson**: Every analysis needs a story
- **Naval Ravikant**: Compress wisdom into memorable formulations
- **Patrick McKenzie**: Reveal hidden systems
- **George Orwell**: Never use long word where short will do

### Anti-AI Fundamentals

- Vary sentence length (2-40 words)
- Include specific personal examples
- State controversial opinions
- Deliberately break rules
- Add emotional variation

## Quick Wins

### Immediate Improvements

1. Delete "I think" and "I believe"
2. Replace "very/really" with stronger words
3. Cut first sentence of paragraphs (often throat-clearing)
4. Add specific numbers/dates
5. Include one controversial opinion

### The 2-35-5 Rule

- Some sentences: 2 words
- Some sentences: 35+ words
- Most sentences: 5-15 words
